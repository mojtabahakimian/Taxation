# پروژه ارسال صورت حساب به سامانه مودیان و استعلام آن

این مخزن شامل مجموعه‌ای از ابزارها برای صدور، ارسال و پیگیری صورت‌حساب‌های الکترونیکی به سامانه مودیان است. ابزارها با زبان C# و پلتفرم .NET 6 توسعه یافته‌اند.

## ساختار مخزن

- **Prg_Moadian**: هستهٔ کتابخانه‌ای که توابع اصلی برای اتصال به سامانه مودیان و ارسال اطلاعات را فراهم می‌کند.
- **Prg_Graphicy**: رابط کاربری گرافیکی (WPF) برای ارسال یا لغو صورت‌حساب توسط کاربر.
- **Prg_Grpsend**: ابزار ارسال گروهی صورت‌حساب‌ها به سامانه مودیان.
- **Prg_TrackSentInvoice**: برنامهٔ پیگیری وضعیت صورت‌حساب‌های ارسال‌شده و مشاهدهٔ خطاهای احتمالی.

## پیش‌نیازها

- .NET SDK 6.0 یا بالاتر
- بستهٔ `TaxCollectData.Library` نسخهٔ 0.0.24 (در پوشهٔ `Prg_Grpsend/InfoDocuments` موجود است)

## نحوهٔ ساخت

```bash
dotnet restore TaxationApp.sln
dotnet build   TaxationApp.sln
```

> **نکته:** در صورت بروز خطا دربارهٔ بستهٔ `TaxCollectData.Library`، ابتدا باید این بسته را از فایل زیپ موجود در پوشهٔ `Prg_Grpsend/InfoDocuments` نصب کرده یا نسخهٔ موردنیاز را در مخزن محلی خود قرار دهید.

## منابع تکمیلی

تمامی مستندات و فایل‌های راهنما در مسیر `Prg_Grpsend/InfoDocuments` قرار دارند.

## راهنمای کدنویسی

- از قواعد نام‌گذاری PascalCase برای کلاس‌ها و متدها استفاده کنید.
- منطق کسب‌وکار را در کتابخانهٔ `Prg_Moadian` نگه دارید و در رابط‌های کاربری فقط به آن ارجاع دهید.
- قبل از ثبت تغییرات، با اجرای `dotnet build` از صحت کامپایل اطمینان حاصل کنید.
