# پروژه ارسال صورت حساب به سامانه مودیان و استعلام اون

- Prg_Graphicy : یه پنجره بصری برای اینکه کاربری که از طریق نرم افزار دیگه این برنامه را صدا زده بتونه ارسال یا لغو کنه به سامانه صورت حسابش رو

- Prg_Grpsend : ارسال گروهی فاکتور ها (صورت حساب ها) به سامانه مودیان

- Prg_Moadian : هسته اصلی یعنی توابع اصلی که عملیات ارسال به سامانه مودیان را انجام میدهند

- Prg_TrackSentInvoice : پیگیری وضعیت صورت حساب های ارسال شده (آیا موفق بوده یا خطا داشته ؟)
